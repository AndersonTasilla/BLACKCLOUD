## 📎 Personalización

Si deseas modificar:

- El **fondo**, cambia la `background-image` en el `body`.
- El **nombre del gremio**, edítalo en el `<header>`, `<title>`, y `<footer>`.
- El **enlace de Discord**, actualiza el `href` del botón `.discord-button`.
